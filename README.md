# 人生脚本 - Life Script App v2

一个基于 Web 的时间管理应用，帮助你像导演一样规划人生剧本。

## 最新版本说明

**版本：** v2.0.0
**最后更新：** 2026-02-14

## 核心功能

- **周计划**：规划每周7天、每天17小时（7:00-23:00）的时间分配
- **四象限分类**：健康习惯、家庭时间、工作时间、自我成长
- **快捷排班**：选择类别和标签后，点击格子快速填充
- **今日打卡**：每天记录完成情况，支持完成/未完成标记
- **周复盘**：AI 深度洞察、知行合一率、时间分布分析
- **分享功能**：生成精美分享卡片

## 技术栈

- 纯 HTML + CSS + JavaScript
- 数据存储：LocalStorage
- 使用 html2canvas 生成分享图片
- 支持 DeepSeek/智谱 AI 分析

## 本地运行

直接在浏览器中打开 `index.html` 即可使用。

## 部署

### 部署到 Vercel

1. 推送代码到 GitHub
2. 访问 https://vercel.com/new
3. 导入 GitHub 仓库
4. 点击 Deploy

### 部署到 Netlify

1. 访问 https://app.netlify.com/drop
2. 拖拽整个文件夹到页面上
3. 等待部署完成

## 数据说明

所有数据存储在浏览器的 localStorage 中：
- 计划数据
- 打卡记录
- 复盘笔记
- API 配置

⚠️ **重要**：更换浏览器或清除缓存会丢失数据，建议定期导出数据备份。
